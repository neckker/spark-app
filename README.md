# ✨ Spark

**Spark** is a desktop app that filters live token launches from
[pump.fun](https://pump.fun) and [bonk.fun](https://bonk.fun), and instantly
opens the matches in your trading terminal.

Join us on [Telegram](https://t.me/neckkero) and
[Discord](https://discord.gg/kzpyEHUdpj), or visit the
[website](https://spark-bot.fun).

## 🎯 Features

- 📡 **Real-time feed** of every new token
- 🎛️ **Smart filters**: dev holdings, funding, migration & fees
- 👁️ **Dev trackers**: whitelist or blacklist wallets
- 🚀 **Auto-open** matches in [Axiom](https://axiom.trade/@neckker), [Padre](https://trade.padre.gg/rk/neckker) or [GMGN](https://gmgn.ai/r/nekky)
- 🖥️ Available for **Windows** and **macOS**
